=================== Some symfony2 command ===================
Setup resources link
rm -r app/cache/*
php app/console cache:clear --env=prod
php app/console assets:install --symlink web
php app/console assetic:dump --env=prod

Generate entity:
php app/console doctrine:mapping:import --force GaoC5Bundle yml
php app/console doctrine:mapping:convert annotation src --force
php app/console doctrine:generate:entities GaoC5Bundle --no-backup

php app/console doctrine:mapping:import --force GaoAdminBundle yml --filter="Admin"
php app/console doctrine:mapping:convert annotation src --force
php app/console doctrine:generate:entities GaoAdminBundle --no-backup

Update to database
php bin/console doctrine:schema:update --force
